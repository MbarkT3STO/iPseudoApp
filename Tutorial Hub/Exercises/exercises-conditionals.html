Exercise 1: Weather Check

Easy • 5 min

Problem:

Check if it's raining and tell the user whether to bring an umbrella or not.

Requirements:

  • Ask user if it's raining (yes/no)
  • Use simple IF statement
  • Display appropriate message

Sample Input/Output:

Sample Run
Is it raining? (yes/no): yes Bring an umbrella!

Solution 

Algorithm WeatherCheck
    var isRaining = Input "Is it raining? (yes/no):"
    
    If isRaining == "yes" Then
        Print "Bring an umbrella!"
    Endif
Endalgorithm
Key Concepts:
  • Simple IF statement for single condition
  • String comparison with == operator
  • Conditional execution based on user input
