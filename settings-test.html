Hardware Acceleration Setting Test

How to test:
  1. Open the iPseudo app
  2. Click the Settings button (gear icon) in the top navigation
  3. Go to the "Advanced" tab
  4. Look for "Hardware Acceleration" toggle at the bottom
  5. Toggle it on/off and observe the changes

What to expect:

  • When ENABLED: Smooth animations, GPU acceleration, better performance
  • When DISABLED: CPU rendering, potentially slower but more compatible

Setting Location:

Settings Modal → Advanced Tab → Hardware Acceleration

✅ Setting should be visible in the Advanced tab of the settings modal
✅ Setting should be checked by default (enabled)
✅ Toggle should work immediately without restart

Technical Details:

  • Setting ID: hardwareAcceleration
  • Default value: true
  • Storage: localStorage as part of iPseudoSettings
  • Affects: All UI elements, Monaco editor, animations, scrolling
