🚀 3-STEP QUICK START:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Convert Icons
───────────────────────

  On Mac/Linux:
    cd app-icons
    ./convert-icons.sh

  On Windows:
    cd app-icons
    convert-icons.bat

  📌 Requirement: ImageMagick must be installed
     Mac:     brew install imagemagick
     Windows: choco install imagemagick
     Linux:   sudo apt-get install imagemagick


STEP 2: Copy to Build Folder
───────────────────────────────

    mkdir -p ../build
    cp windows/ipseudo-icon.ico ../build/icon.ico
    cp macos/ipseudo-icon.icns ../build/icon.icns
    cp linux/ipseudo-icon.png ../build/icon.png


STEP 3: Update package.json
──────────────────────────────

  Add this to your package.json:

  {
    "build": {
      "appId": "com.mbark.ipseudo",
      "mac": { "icon": "build/icon.icns" },
      "win": { "icon": "build/icon.ico" },
      "linux": { "icon": "build/icon.png" }
    }
  }


📸 PREVIEW YOUR ICON:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Open preview.html in your browser to see the icon at
  different sizes and backgrounds.

    Mac:     open preview.html
    Windows: start preview.html
    Linux:   xdg-open preview.html

🔧 TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ❌ "ImageMagick not found"
     → Install ImageMagick (see Step 1)

  ❌ "Icon not showing in app"
     → Check package.json paths
     → Rebuild app completely
     → Clear app cache

  ❌ "Can't create .icns on Windows"
     → Use online converter: cloudconvert.com
     → Or use: npm i -g electron-icon-builder
